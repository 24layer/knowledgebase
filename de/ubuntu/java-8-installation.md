## Sie möchten Java 8 auf Ihren Server installieren?


Server-Paketliste aktualisieren
``` bash
apt update && apt upgrade -y
```

Java 8 installieren
``` bash

apt install openjdk-8-jre-headless -y
```

Überprüfung der Java-Version
``` bash
java -version
```
