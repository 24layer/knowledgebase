## Sie möchten die Zeitzone Ihres Servers anpassen?


Auflistung aller Zeitzonen
``` bash
timedatectl list-timezones
```

Zeitzone setzen
``` bash
timedatectl set-timezone ZEITZONE
```

Überprüfung der Zeitzone
``` bash
timedatectl
```
